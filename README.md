## Sobre o projeto

Projeto desenvolvido como Desafio de Projeto da [DIO](https://www.dio.me/), explorando o **NotebookLM** para curadoria de fontes, engenharia de prompts e organização do conhecimento sobre computação quântica.

O conteúdo foi separado em arquivos por seção para facilitar navegação e manutenção. Use o índice abaixo.
